## Data Analytics & Business Intelligence Portfolio

This repository contains my data analytics and database projects, focusing on SQL querying, data warehousing, and BI dashboarding to solve real business problems.

### Projects

- **[Cloud Billing System Design](./cloud-billing-system-design)**
<br>Created functional process flowcharts, database blueprints, and complete UI wireframe mockups for an automated cloud invoicing system.

- **[Sports League report](./sports-league-database)**
<br>Designed a MySQL database to manage match schedules and team statistics, using strict foreign keys to prevent data conflicts.

- **[Curtin Law School Recruitment Analytics](./enrollment-insights-analytics)**
<br>Cleaned 11 years of student data and built a Power BI dashboard to fix recruitment cluster risks for the Law School.

- **[Driving School Databases](./driving-school-database)**
<br>Built an automated database with custom lookup tables to track student lesson progress, instructor schedules, and payment statuses.

- **[Taiwan Renewable Energy Market Forecast](./renewable-energy-analytics)**
<br>Combined multi-national energy datasets into a Power BI dashboard with time-series forecasting to track long-term green energy trends.

- **[Warehousing Data Analytics](./warehousing-data-analytics)**
<br>Built an Oracle SQL Star Schema and used advanced Window Functions (`ROW_NUMBER() OVER`) to rank carrier performance and catch cost leaks.
