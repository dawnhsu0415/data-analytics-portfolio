## Data Analytics & Business Intelligence Portfolio

Welcome to my data analytics repository. Here, you will find practical projects showing how I clean messy data, build data models, write advanced SQL queries, and create dashboards to help companies solve real-world problems and save money.

### Repository Structure & Projects

#### 1. [Curtin Law School Recruitment Analytics](./enrollment-insights-analytics)
- **What it is:** A real consultancy project analyzing 11 years of historical student enrolment records for Curtin Law School to fix recruitment concentration risks.
- **Tech Used:** Power BI, Power Query, Excel[cite: 11].
- **Key Deliverable:** Cleaned geo-location data, mapped out student origins across Western Australia, and created an executive dashboard showing where the school is missing out on talent[cite: 10, 11].

#### 2. [Taiwan Renewable Energy Market Forecast](./renewable-energy-analytics)
- **What it is:** A market intelligence project cleaning and reconciling multi-national energy datasets to forecast Taiwan's green energy trend up to 2030.
- **Tech Used:** Power BI, Power Query, ETL Pipelines.
- **Key Deliverable:** An interactive executive dashboard with time-series forecasting to show if sustainability targets are realistic.

#### 3. [Corporate Data Warehousing & SQL Analytics](./warehousing-data-analytics)
- **What it is:** A backend project transforming chaotic transactional logistics records into a clean corporate Data Warehouse using Kimball's Star Schema methodology.
- **Tech Used:** Oracle SQL, SQL Developer.
- **Key Deliverable:** Built database structures with strict constraint protections and used advanced SQL Window Functions (`ROW_NUMBER() OVER`) to rank carrier performance and catch shipping cost leaks.

#### 4. [Sports League & Driving School Databases](./database-design-projects)
- **What it is:** Pure database design projects built completely from scratch to show core relational database foundation skills.
- **Tech Used:** MySQL / PostgreSQL.
- **Key Deliverable:** Designed clean Entity-Relationship Diagrams (ERDs), mapped out complex table normalization, and wrote one-click reset scripts using DDL/DML scripts.

#### 5. [Cloud Billing & Infrastructure System Design](./cloud-billing-system-design)
- **What it is:** A business systems analysis project focused on capturing system requirements and mapping out workflows.
- **Tech Used:** Draw.io, Wireframing Tools, Cross-functional Flowcharts.
- **Key Deliverable:** Designed complete system mockups, database blueprints, and UI flow diagrams for a cloud-based automated invoicing system.
